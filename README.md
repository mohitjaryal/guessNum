# 🎯 Guess The Number Game (Python)

## 💡 Overview
**Guess The Number** is a fun and beginner-friendly Python project where the computer randomly selects a number, and the player tries to guess it in as few attempts as possible.  

This project helped me strengthen my understanding of loops, conditional statements, user input handling, and random number generation using core Python concepts — all while building a simple, interactive game.

---

## 🚀 Features
- 🎲 Randomly generates a number within a range  
- 👨‍💻 Takes user input to guess the number  
- 🧠 Gives hints if the guess is too high or too low  
- 🏁 Tracks number of attempts  
- ⚙️ 100% built using **pure Python** — no external libraries  

---

## 🧠 What I Learned
Building this project helped me understand:
- How to use the `random` module in Python  
- Handling user input with validation  
- Using loops and conditionals effectively  
- Implementing simple game logic  
- Writing clean, readable, and interactive Python code  

---

## 🛠️ Tech Stack
- **Language:** Python 🐍  
- **Modules Used:** `random` (built-in)  
- **Environment:** Works in any Python 3.x terminal or IDE  

---
## 🎮 Gameplay Instructions**
1. The computer selects a random number within a given range (e.g., 1–100).  
2. Enter your guess when prompted.  
3. The program will tell you if your guess is too high or too low.  
4. Keep guessing until you find the correct number.  
5. Try to win in the fewest attempts possible!
